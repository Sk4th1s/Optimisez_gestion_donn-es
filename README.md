# Optimisez_gestion_donnees

Analyser les ventes d'une boutique de vin. Détecter les erreurs dans les fichiers

Pour se projet, j'ai commencé par nettoyer les fichiers de données en supprimant les éléments inutiles. J'ai ensuite regroupé toutes mes données dans un seul dataframe.
J'ai ensuite analyser le nombre de ventes par produit, le chiffre d'affaires de chaque produit, le prix des différents produits.
J'ai aussi analyser les produits avec un prix très différents des autres pour voir si c'était une erreur ou juste un produit plus prestigieux.
